LeapRevo


Built in WearHack Toronto 2016
LeapRevo is a desktop rhythm game designed for exercise wrist through playing. The game requires players to twist their wrists following the arrow on the screen. Player can also make their own maps using .osu file and upload them to the server.

  • Client Side: Leap Motion, Unreal Engine, C++
  • Backend & Website: AWS, Node.js, Javascript, Osu-parser
